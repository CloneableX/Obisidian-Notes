### 树形递归

树形递归也是通过替换模型分析产生，树形递归在一个表达式中会进行多次递归调用，形成类似树形的递归计算过程，所以称为树形递归过程，此过程的时间消耗是指数级的，与树形的节点呈正相关性，消耗的空间与树的最大深度呈正相关性。

树形递归存在大量的冗余计算，因此它的计算效率并不高，但它描述及解决某些问题更加直观（例如继承类型的数据），所以它也是一种设计程序的重要工具。

下列计算斐波那契数列的程度为树形递归过程：

```scheme
(define (fib n)
  (cond ((= n 0) 0)
        ((= n 1) 1)
        (else (+ (fib (- n 1))
                 (fib (- n 2))))))
```
