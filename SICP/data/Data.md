基础数据类型包含数字类型和标识类型

### 标识类型

在 Scheme 中数据的基础类型除了数字类型还有标识类型，它可以表示标识符号本身的语法含义，而不是符号产生的计算结果，但是需要注意，数字的标识结果还是数字本身。

通过「'」表示标识

```scheme
(define a 1)
(define b 2)
(list a b)
(1 2)
(list 'a 'b)
(a b)
(list 'a b)
(a 2)
```

单引号加上列表的输出形式表示标识列表：

```scheme
(car '(a b c))
a
(cdr '(a b c))
(b c)
```

同理 `'()` 表示空列表，另外判断两个标识是否相等可以使用基础程式 `eq?` 。
